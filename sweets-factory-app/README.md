# تطبيق مصنع الحلويات

تطبيق Flutter متكامل لإدارة مصنع حلويات وطرد، مرتبط بنظام ERPNext.

## هيكل المشروع

```
lib/
├── main.dart                      # نقطة البداية
├── core/
│   ├── api/
│   │   └── erp_next_service.dart  # خدمة الربط مع ERPNext
│   └── models/
│       ├── product.dart           # نماذج المنتجات
│       ├── order.dart             # نماذج الطلبات
│       └── user.dart              # نماذج المستخدمين والمدفوعات
├── features/
│   ├── auth/
│   │   ├── screens/
│   │   │   └── login_screen.dart # شاشة الدخول
│   │   └── widgets/
│   │       └── login_form.dart    # نموذج الدخول
│   └── orders/
│       └── screens/
│           ├── orders_screen.dart       # قائمة الطلبات
│           ├── new_order_screen.dart    # أمر عمل جديد
│           ├── kitchen_screen.dart      # شاشة المطبخ
│           ├── delivery_screen.dart     # شاشة السائق
│           └── payment_screen.dart      # شاشة المدفوعات
└── shared/
    └── themes/
        └── app_colors.dart         # ألوان التطبيق
```

## المميزات

### 1. شاشة الدخول
- مصادقة عبر ERPNext API
- حفظ الجلسة محلياً
- دعم التحقق من الجلسة المحفوظة

### 2. شاشة الطلبات
- عرض جميع الطلبات مع الفلترة حسب الحالة
- إنشاء طلب جديد
- عرض تفاصيل الطلب

### 3. أمر عمل جديد
- إدخال بيانات العميل
- اختيار المنتجات والكميات
- رفع المرفقات (صور، مستندات)
- تحديد وقت التسليم

### 4. المطبخ
- عرض الطلبات النشطة
- تنبيهات عاجلة حسب وقت التحضير
- تحديث حالة الطلب

### 5. التوصيل
- عرض الطلبات الجاهزة
- اتصال بالعميل
- فتح الخرائط
- تأكيد التوصيل وتحصيل المبلغ

### 6. المدفوعات
- تتبع المبلغ المدفوع والمتبقي
- تسجيل المدفوعات الجديدة
- عرض سجل المدفوعات

## التثبيت
